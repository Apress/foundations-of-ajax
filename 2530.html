Chapter 1

Chapter 2

  1. Page 27, number 1: There should be a space between the type and id attributes. There should also be a semicolon after the validateEmail() JavaScript function call.

  3. Page 27, sixth line from bottom: The text should refer to a context type of text/xml, not text/html.

Chapter 3

  3. Page 59, Listing 3-7: There are three <td> tags that do not have closing </td> tags.

Chapter 4

  1. Page 103, Listing 4-11: The line:
    var parData = "Par: " + par
    in function setData is missing the semicolon at the end.

Chapter 7

  1. Page 198, first line: The text "... for CSS files to text/plain" should read "... for CSS files to text/css."
  2. Page 206, paragraph 4: The Step Out function is incorrectly referred to as Stop Out.

Chapter 8

  1. Page 224: Just before the "Learning Where to Go for More" section, the sentence "Before you role out some snappy new Ajax feature, do some paper mock-ups, ..." should read "Before you roll out...."
  2. Ajax Dashboard example: A couple of the external Web services that the example relies upon for data have stopped functioning, causing the entire application to fail.
